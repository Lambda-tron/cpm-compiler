C+- Compiler

This project is a compiler for a small experimental programming language
called C+-. It was developed as part of a university course project to
demonstrate the main concepts of compiler construction.

The compiler implements the front-end stages of compilation, including:

-   Lexical analysis (Flex)
-   Parsing (Bison)
-   Abstract Syntax Tree (AST) generation
-   Symbol table and scope handling
-   Semantic analysis and type checking

The language C+- is inspired by C/C++ and supports features such as: -
Primitive types (INT, FLOAT, BOOL) - Arrays - Classes and methods -
Control structures (IF, FOR) - Function calls and return statements

The compiler analyzes C+- source files and reports syntax and semantic
errors such as undefined identifiers, duplicate declarations, type
mismatches, and invalid function calls.

This project focuses on the design and implementation of a compiler
front-end for educational purposes.
